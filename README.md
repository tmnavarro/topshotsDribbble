# TOPSHOTS Dribbble

Webapp para requisição dos melhores trabalhos postados no Dribbble

## Ferramentas Utilizadas

Yoeman com AngularJS.
 - Grunt
 - Sass
 - Dribbble API v1


##Veja em funcionamento
[Sistema no online](https://thoshotsdribbble-tnavarro.rhcloud.com)

##Instalação local

  - Faça o clone do projeto para sua máquina
  - Instale as dependências com os comandos:
   - bower install
   - npm install
  - Execute o projeto com Node ou Grunt
   - npm start
   - grunt serve
